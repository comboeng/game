WebGLでページ表示時にuncaught syntaxerror unexpected identifierエラーが表示される

  • Mar 21, 2021
  • エラー詳細
  • 対応策

エラー詳細

WebGLのページを表示しようとしたとき uncaught syntaxerror unexpected identifier と表示され、ゲームのロードがストップし、ゲームをプレイすることができない現象がありました。

対応策

[SOLVED] Unity 2020 WebGL Doesn’t work Uncaught SyntaxError: Invalid or unexpected token

上記ページ内の対応を行いました。

具体的には WebGLビルド時に出力される、index.htmlの１行目に

<meta http-equiv="content-encoding" content="gzip" />
を追記しました。

それでも治らない場合は

Project SettingsのPlayerのCompression FormatをDisabledにすると良いかもしれません。
